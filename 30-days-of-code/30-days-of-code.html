30 DAYS OF CODE

A coding challenge inspired by Jennifer Dewalt where I am going to code every day for a month starting July 21, 2017.

The aim of this challenge is to learn a new design trick everyday for the next thirty days. If you want to do this challenge, feel free to do so! This is not for complete beginners in web design but for people with basic knowledge of HTML, CSS and JavaScript.

DAY 1

21 July 2017

THE PARALLAX SCROLLING EFFECT

Parallax is an effect where the background content or image in this case, is moved at a different speed than the foreground content while scrolling.

DAY 2

22 July 2017

3D HOVER PLANE EFFECT

The element seems to be moving back and forth while hovering over the screen.

DAY 3

23 July 2017

PAPER CUT-OUT EFFECT

Text effect which appears like a paper cut-out.
